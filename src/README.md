# ⚠️ RETIRED — legacy Svelte frontend (do not submit PRs here)

This tree is the pre-2.0 Svelte UI. **It no longer ships**: since QBZ 2.0 the
app is a single native Rust process with a Slint UI living under
[`crates/`](../crates/). This code remains in the repository only as a
porting reference and will be removed in an upcoming cleanup release.

If you found a bug here, it very likely exists (or was already fixed) in the
live app — please target `crates/` instead, or open an issue and we will help
you port the change.
