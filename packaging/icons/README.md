# Canonical app icons

Canonical icon source for every Linux/macOS packager (nfpm deb/rpm, AppImage,
tarball, flatpak, snap, ebuilds, PKGBUILDs). Copied out of `src-tauri/icons/`
on 2026-07-02 so packaging stops depending on the retired Tauri tree —
`src-tauri/` is reference-only and will eventually be deleted.

- `NNxNN.png` — hicolor sizes (32/48/64/128/256/512, plus `128x128@2x`)
- `icon.icns` — macOS bundle icon

If the artwork ever changes, update BOTH copies (or delete the src-tauri one).
